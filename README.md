# Hotel-Management-System
This project is a Hotel Management System designed to handle various administrative and operational aspects for a hotel, specifically tailored for "The Taj Group" as seen in the interface.

Purpose and Functionality
The system manages hotel reception activities and supports staff in tasks such as customer check-in/check-out, room assignment, employee management, service requests, and status updates.

Key Features
New Customer Form: Registers new guests and gathers their information for check-in.

Rooms/Department Management: Assigns rooms and manages different hotel departments.

Employee Management: Keeps track of all employees and their roles within the hotel.

Customer Info & Manager Info: Provides details about guests and managerial staff for easy access and review.

Checkout/Update Status: Facilitates smooth guest checkout and updates the status of rooms and guests.

Pickup Service: Coordinates guest transportation and special services.

Search Room/Logout: Helps staff quickly locate available rooms and securely exit the system interface.

User Interface
The interface has a visually appealing background representing a luxury hotel, with a navigation panel for reception tasks and a central section displaying "HOTEL RECEPTION" for clarity and branding.

Overall, it is a hotel management software focusing on front desk operations that streamline guest services and internal hotel administration.
